## 💻 Desarrollo de software
- Desarrollo de aplicaciones de escritorio con núcleo científico y de simulación (Python, C++, PyTorch) para modelado de sistemas complejos, dinámica estocástica y mecánica estadística computacional.
- Desarrollo de aplicaciones web y móviles progresivas (PWA) con React, Next.js y TypeScript, optimizadas para experiencias de usuario fluidas y alto rendimiento en dispositivos móviles.
- Desarrollo de sitios web institucionales, promocionales y catálogos interactivos con arquitecturas estáticas y dinámicas de alto rendimiento (Tailwind CSS, GSAP, Cloudflare Pages).
- Desarrollo de sistemas web empresariales full-stack con FastAPI, Node.js, Express, bases de datos SQL/NoSQL y orquestación de microservicios (Docker, Kubernetes, AWS).

## 🔧 Personalización de software
- Personalización y fine-tuning de modelos de IA (YOLOv8, Transformers, LLMs locales como Llama) para detección de objetos, clasificación médica, análisis de sentimiento y generación de lenguaje natural adaptados a datos propietarios.
- Creación de plugins, agentes autónomos de negociación (LangGraph) y pipelines RAG completos con chunking recursivo, recuperación semántica y almacenamiento vectorial (ChromaDB, Sentence Transformers).
- Mejoras integrales en el rendimiento y la escalabilidad: optimización de consultas SQL, implementación de caché distribuida con Redis, contenerización con Docker y orquestación en la nube con Kubernetes y AWS.
